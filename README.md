# Diabetes-Prediction-using-Support-Vector-Machines


This project focuses on predicting the likelihood of diabetes using Machine Learning techniques, specifically the Support Vector Machine (SVM) algorithm. By analyzing medical attributes such as glucose level, blood pressure, BMI, insulin level, and age, the model identifies patterns associated with diabetes risk. The system includes data preprocessing, feature scaling, model training, and performance evaluation to ensure accurate predictions. The goal is to provide an efficient and reliable tool that can assist healthcare professionals in early diabetes detection and support preventive healthcare decisions.

Tech Stack: Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Support Vector Machine (SVM).
